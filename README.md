## Exform: Form Application

## Run Locally

Clone the project

```bash
  git clone git@github.com:umarhadimukti/exform-backend.git
```

Go to the project directory

```bash
  cd exform-backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  pnpm run start
```

## Development


```bash
  pnpm run dev
```
